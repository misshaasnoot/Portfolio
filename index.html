Rent a Pet
Previous Next

Rent a Pet is a prototype for a online pet rental platfrom with a structure similar to that of AirBnb's, created for the purposes of practice. Users can view, search for and request to rent a pet for selected dates. They can also create their own listings and accept or reject booking requests from other users. This website is created using HTML, CSS and Ruby on Rails.

JuniorHub is a job platform created to help companies find and hire junior developers, while also allowing the latter to advertise their services. The platform has an application system, direct messaging function, search function and personal profile creation. It is created using HTML, CSS, JavaScript and Ruby on Rails.
